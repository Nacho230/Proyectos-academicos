# 🎓 Portfolio de Proyectos Académicos

Bienvenido a mi repositorio de prácticas y trabajos universitarios. Este espacio recopila diversos proyectos desarrollados a lo largo de mi formación en la carrera de Ingeniería en Informática. 

El objetivo de este repositorio es reflejar mi evolución técnica, la comprensión de distintos paradigmas de programación y la aplicación lógica de algoritmos en lenguajes como **Python** y **C#**.

---

## 📂 Proyectos Incluidos

### 1. Inmobiliaria Integral "Javac" (Python)
**Directorio:** `Proyecto final cursada python/`

Sistema de gestión desarrollado como Trabajo Práctico Final Integrador para la materia "Fundamentos de Programación". La plataforma administra el catálogo de una inmobiliaria, gestionando propiedades, clientes y contratos.

* **Características:**
  * ABM de propiedades (venta/alquiler) y cartera de clientes.
  * Generación y administración de contratos y fechas de firma.
  * Uso intensivo de diccionarios y listas para el almacenamiento y filtrado de datos en memoria.
* **Demostración en Video:** ▶️ [Ver video explicativo del proyecto](https://www.youtube.com/watch?v=gt4zagf1HZ8)

### 2. Sistema de Telefonía Móvil (C#)
**Directorio:** `Proyecto-Facultad/`

Aplicación de consola diseñada para simular la gestión de clientes y líneas de una compañía telefónica. Este proyecto demuestra el dominio del paradigma de la **Programación Orientada a Objetos (POO)** y la arquitectura de software.

* **Características:**
  * Lógica transaccional simulada (llamadas, mensajes, recargas y transferencias de saldo).
  * Control de estado lógico de las entidades (líneas activas/suspendidas).
* **Conceptos técnicos destacados:**
  * Diseño de clases y diagramado UML.
  * Creación y captura de **Excepciones Personalizadas** (`SaldoInsuficienteException`, `ClienteSinLineasException`).
  * Manejo de colecciones genéricas (`List<T>`).
